## Data Points

| Field Name | Description |
|------------|-------------|
| `url` | 상세 정보를 위해 선수의 프로필 페이지로 직접 연결되는 ESPN URL입니다. |
| `player_name` | NBA 선수의 전체 이름입니다. |
| `player_games_played` | 시즌 동안 해당 선수가 출전한 총 경기 수입니다. |
| `player_games_started` | 시즌 동안 해당 선수가 선발로 출전한 총 경기 수입니다. |
| `player_minutes_per_game` | 해당 선수가 경기당 평균 출전한 시간(분)입니다. |
| `player_points_per_game` | 해당 선수가 경기당 평균 득점한 점수입니다. |
| `player_offensive_rebounds__per_game` | 경기당 평균 공격 리바운드 수입니다. |
| `player_defensive_rebounds_per_game` | 경기당 평균 수비 리바운드 수입니다. |
| `player_rebounds_per_game` | 해당 선수가 경기당 평균 기록한 총 리바운드 수입니다. |
| `player_assists_per_game` | 해당 선수가 경기당 평균 기록한 어시스트 수입니다. |
| `player_steals_per_game` | 해당 선수가 경기당 평균 기록한 스틸 수입니다. |
| `player_blocks_per_game` | 해당 선수가 경기당 평균 기록한 블록 수입니다. |
| `player_turnovers_per_game` | 해당 선수가 경기당 평균 기록한 턴오버 수입니다. |
| `player_fouls_per_game` | 해당 선수가 경기당 평균 기록한 파울 수입니다. |
| `player_assist_to_turnover_ratio` | 해당 선수의 어시스트 대비 턴오버 비율입니다. |
| `team` | 시즌 동안 해당 선수가 소속된 NBA 팀입니다. |
| `season_type` | 데이터가 정규 시즌인지 플레이오프인지 여부를 나타냅니다. |
| `season_year` | 통계가 해당하는 NBA 시즌의 연도입니다. |


## Use Cases

### 1. Player Performance Analysis
여러 시즌에 걸친 개별 선수의 성과를 분석하여 상위 성과자를 식별하고 팀 전략을 개선합니다.

### 2. Fantasy Basketball Insights
과거 선수 통계를 활용하여 판타지 농구 리그를 위한 인사이트를 생성하고, 선수 선택 및 예측에 도움을 제공합니다.

### 3. Sports Journalism and Reporting
스포츠 기사에 심층적인 통계 분석을 제공하여, 선수 성과에 대한 데이터 기반 인사이트로 보도를 강화합니다.
